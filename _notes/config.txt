1. Инициализируем проект: npm init -y ( создается файл package.json, файл с информацие о проекте и зависимостями в проекте).
2. Добавляем общие настройки среды разработки для проекта: editorconfig ( Устанавливаем расширение для VSCode ).
3. Инициализируем Git в проекте: git init.
4. Скрыть личные файлы от удаленного git репозитория, добавляем файл gitignore.
5. Настраиваем линтер для соблюдения общих правил написания: eslintrc.json ( Устанавливаем расширение ES Lint).
6. Настраиваем линтер для типизации, для этого добавляем файл tsconfig.json и устанавливаем typescript в проект как зависимость: npm instal typescript
7. Заполняем пользовательский файл о проекте - README.md
